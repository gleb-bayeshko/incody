### Необходимые зависимости:

> - `node    v20.16.0+`
> - `npm     v10.8.1+`

### Установка зависимостей:

```bash
npm install
```

### Запуск dev версии

```bash
npm run dev
```

Приложение запустится на `http://localhost:5173`.

### Сборка билда

```bash
npm run build
```

### Билд

Итоговый билд будет находится в папке build/client
